# MobyTitle — liner-note титр (регистр ранних клипов Moby)

Плоский 2D-типографический слой под бриф v3 §4 (пресет C): `yaromat` строчными
крупнее + название трека мельче приглушённым + тонкая hairline-линейка. Медленный
fade-in, микро-дрейф вверх, тихий разъезд линейки. Альфа-композит (ProRes 4444).

**Якорь:** графический язык ранних клипов Moby (Play-эра) — подпись как на обороте
пластинки, а не CapCut-хук. Заземлён на реальный референс (лечит «дёшево из генерации»).

**Пропсы:** `accentText` = название трека (опц.). Палитра приглушённая, без неона.
